# dillisp
Lisp implemented in Python. Created by Dillon Beliveau (GET IT?)

## Dependencies
* Python 2.x (Although it should work in 3 with no problems)

## Features
* Lexical Scope
* First class functions
* REPL and script support
* Tail-call optimization
